## Installation

1. Clone the repository:
git clone https://github.com/PRATHAMTEJANI/HexSoftwares_Voice_assistant.git

2. Navigate to the project folder:


cd HexSoftwares_Voice_assistant

3. Install required dependencies:


pip install -r requirements.txt

4. Create a `.env` file in the project root and add your OpenAI API key:


OPENAI_API_KEY=your_api_key_here


## Usage

Run the assistant using:


python your_script_name.py

Speak commands like:
- "What time is it?"
- "Search Wikipedia for Python programming"
- "Play Despacito on YouTube"
- "Tell me a joke"
- "Open Google"

Say "exit", "stop", or "quit" to close the assistant.

## Important Notes

- Keep your **OpenAI API key secure**. Do not push it to public repositories.
- Requires an active internet connection for AI responses and online searches.
